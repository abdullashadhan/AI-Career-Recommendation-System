# AI-Career-Recommendation-System
A Streamlit-based web application that recommends career paths using Natural Language Processing (NLP) and machine learning.

Technical Overview:

•Text preprocessing and feature extraction using TF-IDF
•Career classification using a Decision Tree model
•Similarity-based insights using cosine similarity
•Interactive UI built with Streamlit

Features:

•Skill and interest-based career recommendation
•NLP text processing using TF-IDF
•Machine learning model using Decision Trees
•Interactive web interface built with Streamlit
•Lightweight and easy to run locally

Tech Stack:

•Python
•Pandas, NumPy
•Scikit-learn
•Streamlit

How to Run:
pip install -r requirements.txt
streamlit run app.py

Use Case:
Career guidance system for students and AI/ML portfolio demonstration.
